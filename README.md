# Casper AgentVault

**The unified smart wallet for agentic DeFi and RWA on Casper Network**

One wallet. Three powerful agentic modules working together.

## Features

### 1. Portfolio Guardian
Autonomous DeFi yield optimizer and risk sentinel. Monitors positions, rebalances safely, explains decisions.

### 2. RWA Compliance Oracle
AI-powered agent that fetches real-world data, runs compliance/risk assessment, posts verifiable on-chain attestations with reputation.

### 3. Agent Marketplace & Trust Layer
Escrow-powered marketplace for AI agents. Hire, verify work, build on-chain reputation via x402 payments.

## Tech Stack
- **Blockchain**: Casper Network (Odra smart contracts, x402, MCP)
- **Agents**: Multi-agent system with LLM reasoning
